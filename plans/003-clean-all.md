## Why this matters

The CLI surface is asymmetric: users can clean the cache for one app
(`go clean app.cs`) but the only way to clean everything is the `cleanup`
command, which is `[Hidden]` (it exists for the background scheduler and only
deletes *stale* directories). There is no discoverable way to say "wipe all of
go's caches now" — useful when reclaiming disk space or recovering from a
corrupted cache. This plan adds `go clean --all`, reusing the existing sweep
with a 0-day cutoff (delete everything, regardless of age).

## Steps

### Step 1: Extend the `clean` command

In `src/go/Program.cs`, change `CleanAsync` so `input` is optional and an
`--all` switch is available. Exact shape:

```csharp
/// <summary>
/// Deletes cached publish artifacts for a file-based .NET app.
/// </summary>
/// <param name="input">Path to an existing .cs file.</param>
/// <param name="all">Delete cached artifacts for all apps instead.</param>
static int CleanAsync([Argument] string? input = null, bool all = false)
{
    if (all)
    {
        if (input is not null)
        {
            ConsoleApp.LogError("Specify either a .cs file or --all, not both.");
            return 1;
        }

        return CleanupManager.Cleanup(days: 0);
    }

    if (input is null)
    {
        ConsoleApp.LogError("Specify a .cs file to clean, or --all to clean all cached apps.");
        return 1;
    }

    if (!TryResolveEntryPoint(input, out var publishDir, out var stamp))
        return 1;

    return AppCleaner.Clean(publishDir, stamp);
}
```

Why `Cleanup(days: 0)` works: the sweep deletes directories whose last write
time is older than `DateTime.UtcNow.AddDays(-days)`; with `days: 0` the cutoff
is "now", so every cache directory qualifies. After plan 001 the sweep is
best-effort (locked directories are skipped, not fatal) and records
`LastCleanupUtc`, which harmlessly also resets the background-cleanup clock.

**Verify**: `dotnet build` → exit 0, and `git diff --stat src/go/help.md`
shows the regenerated help now lists the `--all` option for `clean`.
